Парсит уязвтимые драйверы в выбранной папке и генерирует заголовок с метками времени.

Зачем? Для детекта определенных драйверов. Юзер может переименовать файл, но таймстамп он не изменит - слетит подпись.

При запуске на ЛКМ софт будет парсить из ```assets\\```. Также можно вручную выбрать папку:
```commandline
main.exe <папка с файлами>
```
